## Mô tả project
Website landing page cho một startup / sản phẩm công nghệ với giao diện hiện đại,  
tập trung vào **UI/UX**, animation mượt và khả năng tương tác tốt.

Project gồm **4 trang riêng biệt**:
- Home
- Features
- Pricing
- Contact

Trang web được thiết kế **responsive**, hoạt động tốt trên desktop, tablet và mobile.

---

## Các chức năng chính
- Header sticky, menu highlight trang hiện tại
- Mobile menu có animation
- Hero section full màn hình
- Layout sử dụng CSS Grid & Flexbox
- Animation hover và animation theo scroll
- Slider testimonial tự động
- Accordion FAQ
- Form liên hệ:
  - Floating label
  - Validation cơ bản
  - Toast thông báo
  - Lưu dữ liệu vào localStorage
